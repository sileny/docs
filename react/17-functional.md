# functional

函数式编程

```jsx
class App extends React.Component {
  render() {
    return (
      <div class="app">
        <h1 class="title">title</h1>
        <p>content</p>
      </div>
    )
  }
}
```
函数式编程的写法如下，

```js
function App(props) {
  return (
    <div class="app">
      <h1 class="title">title</h1>
      <p>content</p>
    </div>
  )
}
```
